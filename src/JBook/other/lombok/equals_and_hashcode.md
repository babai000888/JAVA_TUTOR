### Equals And Hashcode

* Зачем нужен `hashcode` и что это такое - link
#### Использование по умолчанию
При использовании аннотации `@EqualsAndHashCode`:
```java
@EqualsAndHashCode
public class Item {
    private Long id;
    private String name;
    private List<String> providers;
}
```
